# 🧠 CHEAT SHEET — jak rozpoznać pattern w 10 sekund

## 🟦 Graph (BFS/DFS)
**Sygnały:**
- grid / wyspy / regiony
- connected components
- path exists

**Użyj:**
- BFS → shortest path
- DFS → exploration

---

## 🟩 Sliding Window
**Sygnały:**
- substring / subarray
- longest / shortest
- at most K

**Użyj:**
- left + right pointer + hashmap

---

## 🟨 Two Pointers
**Sygnały:**
- sorted array
- pairs / sums

---

## 🟧 Prefix Sum + HashMap
**Sygnały:**
- subarray sum = k
- count subarrays

---

## 🟥 Intervals
**Sygnały:**
- overlap
- merging ranges

---

## 🟪 Monotonic Stack
**Sygnały:**
- next greater
- histogram
- sliding max

---

## ⚫ Heap
**Sygnały:**
- top K
- merging sorted
- median

---

## ⚪ Binary Search
**Sygnały:**
- sorted
- min/max possible

---

## 🟫 Dynamic Programming
**Sygnały:**
- decisions
- number of ways

---

## 🔵 Backtracking
**Sygnały:**
- combinations
- permutations

---

## 🟢 Union-Find
**Sygnały:**
- cycles
- connectivity

---

## 🟡 Design
**Sygnały:**
- implement class
- O(1) operations
